h# Head and Meta Tags

This section covers the head tag and meta tags

## What Problem Does This Solve?

Optimizes web pages for the browser

## Definition

The head tag contains information not displayed in the browser. It contains all the data about the page like its title, character encoding, and responsive rules.

## Mental Model

## Syntax/Structure

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>A simple document</title>
    <link rel="stylesheet" href="style.css" />
  </head>
  <body>
    <!-- Any content goes here -->
  </body>
</html>
```

## Example

**Code**

```html

```
