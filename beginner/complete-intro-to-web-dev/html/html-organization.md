# HTML Organization

This section covers how to organize an HTML document.

## What Problem Does This Solve?

This reduces the need to repeat yourself when writing html. We could create a component using tags and classes structured in a way that we can copy and paste it wherever we need and the same styles and structure will be applied to it.

## Syntax/Structure

```html
<article class="social-post">
  <h2 class="user-name">@jaycobgomez</h2>
  <h3 class="posted-date">Posted 5m ago</h3>
  <img
    class="profile-picture"
    src="url.jpg"
    alt="a laptop screen showing a javascript function"
  />
  <p class="social-post-text">
    Today I wrote my own JavaScript function for the first time that prints
    "hello, world" to the console 3 times!
  </p>
</article>
```
