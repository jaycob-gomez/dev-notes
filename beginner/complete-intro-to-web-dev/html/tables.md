# HTML Tables

This section covers HTML tables.

## What Problem Does This Solve?

In the case when one needs to display stats of any kind, tables are great for displaying this data into a table format.

## Definition

HTML tables store tabular data. It sets up columns and rows and uses other more specific tags to specify headings and group the table data together.

## Mental Model

## Syntax/Structure

```html
<table>
  <!-- Groups the column headings -->
  <thead>
    <!-- Set up rows using table rows -->
    <tr>
      <!-- Table data will create columns -->
      <td>Heading 1</td>
      <td>Heading 2</td>
    </tr>
  </thead>
  <!-- Groups the table data -->
  <tbody>
    <tr>
      <td>3</td>
      <td>4</td>
    </tr>
    <tr>
      <td>5</td>
      <td>6</td>
    </tr>
  </tbody>
</table>
```
